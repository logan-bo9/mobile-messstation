# 📡 Mobile Messstation

## Projektbeschreibung
Dieses Projekt ist eine Arduino-basierte mobile Messstation zur Erfassung von Umweltparametern wie Temperatur und Luftfeuchtigkeit. Die Daten werden auf einem LCD-Display angezeigt und können später erweitert werden (z. B. Warnsysteme, Funkübertragung).

## 🔧 Hardware
- Arduino UNO R4 / kompatibel
- DHT11 / DHT22 Sensor
- LCD 16x2 I2C Display
- Buttons für Menüsteuerung

## 🎯 Ziele
- Echtzeit-Messung von Umweltdaten
- Benutzerfreundliche Anzeige
- Erweiterbares modulares System

## 🧠 Funktionen (geplant)
- Temperaturanzeige
- Luftfeuchtigkeit
- Menüsystem
- Alarmfunktion bei Grenzwerten
