📡 Mobile Messstation

Embedded Systems Projekt – Umweltmessung mit Arduino

🎯 Projektziel

Entwicklung einer mobilen Messstation zur Erfassung von Temperatur und Luftfeuchtigkeit mit Echtzeit-Anzeige auf einem LCD-Display.

⚙️ Technische Komponenten

Arduino UNO R4
DHT22 Sensor
LCD 16x2 I2C
Buttons
Breadboard

🧠 Funktionsweise

Sensoren erfassen Umweltdaten → Arduino verarbeitet Signale → Ausgabe erfolgt auf LCD Display in Echtzeit.

🏗️ Systemarchitektur

🌡 DHT22 → 📟 Arduino → 📺 LCD Display

🚧 Projektstatus

🟡 In Entwicklung – Prototyp Phase
