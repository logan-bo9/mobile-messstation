# 📡 Projektbeschreibung

Die mobile Messstation ist ein Arduino-basiertes System zur Erfassung von Umweltdaten.

## Funktionsprinzip
Sensoren erfassen Temperatur und Luftfeuchtigkeit und geben diese an den Arduino weiter. Die Daten werden auf einem LCD angezeigt.

## Erweiterungen (geplant)
- Menüsystem
- Alarmfunktion bei Grenzwerten
- Speicherung von Daten
